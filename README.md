# Simple Interest Calculator

A simple interest calculator built with HTML, CSS, and JavaScript.
This project was developed as part of the IBM Skills Network coursework to practice core web development concepts like DOM manipulation, input validation, and basic styling.

## Features

- Calculates simple interest based on user input

- Validates principal to ensure it is a positive number

- Displays total interest and final amount

- Automatically shows the future year of maturity

- Clean and easy to understand interface

## Technologies Used

- HTML5

- CSS3

- JavaScript (Vanilla JS)

## How to Use

1. Enter the principal amount.

2. Select the interest rate.

3. Enter the number of years.

4. Click the Compute Interest button.

5. The calculator displays:

> - Amount deposited
> - Interest rate
> - Calculated interest
> - Maturity year
> - Total amount

### Sample Calculation Formula

Simple Interest formula used:

```bash
Interest = (Principal × Rate × Years) / 100
Amount = Principal + Interest
```
